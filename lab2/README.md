## Вывод

В ходе лабораторной работы были сгенерированы бинарные последовательности длиной 128 бит с помощью стандартных генераторов случайных чисел языков C++ и Java. Полученные последовательности были проверены тремя статистическими тестами NIST.

По результатам тестирования все значения p-value оказались больше 0.01, что означает успешное прохождение тестов. Следовательно, сгенерированные последовательности можно считать достаточно случайными в рамках проведённого анализа.
